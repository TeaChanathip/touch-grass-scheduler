# Background
**Note: This project is under the development.**

Touch Grass Scheduler (TGS) is a web application for teachers to systematically assign homework to students inspried from SCRUM principle.

This application will prevent the teachers to separately assiging homework at a time that might exceed students's capabilities. It will help student to have more free time to spend on what thier love like hobbies.

Additionally, the reason I initiated this project is to enhance my software engineering skills on both coding and system design skills.

# Tech Stack
- Frontend: Next.js 15
- Backend: Gin Web Framework + uber-go/fx (Dependency Injection)
- Database: PostgreSQL 18 + pg_cron
- Object Storage: Minio
- Container Engine: Docker
